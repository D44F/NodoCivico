# Nodo Cívico

Aplicación móvil Android orientada a la gestión de reportes ciudadanos dentro de un entorno comunitario, como barrios, conjuntos residenciales, campus o sectores urbanos.

El proyecto permite registrar situaciones del entorno, consultar reportes, revisar su estado y navegar entre diferentes pantallas funcionales mediante fragments.

## Descripción general

Nodo Cívico es una aplicación móvil diseñada para facilitar la creación y seguimiento de reportes comunitarios. Su propósito es apoyar la gestión de incidencias locales, como problemas de alumbrado, aseo, servicios, seguridad o zonas comunes.

En esta primera entrega se presenta la estructura base de la aplicación, la navegación entre fragments y una interfaz inicial funcional con datos simulados.
